### 第二轮：增加旋转/透明渐变/大小渐变
- PathArrayCycler.cs 新增功能：
  - **旋转**：perInstanceRotation (Euler偏移) + rotationSpeed (自旋速度 °/s)，叠加在路径朝向之上
  - **透明渐变**：enableAlphaGradient + alphaOverPath (AnimationCurve)，横轴路径位置→纵轴Alpha
    - 支持 MeshRenderer（_Color属性）和 ParticleSystem（startColor）
    - 缓存 instanceRenderers 避免每帧 GetComponent
  - **大小渐变**：enableScaleGradient + scaleOverPath (AnimationCurve)，横轴路径位置→纵轴缩放倍率
- Moshi_PathArrayWindow.cs 新增 UI：
  - 旋转设置（可折叠）：偏移旋转 Vector3Field + 自旋速度 Slider (-720~720)
  - 透明渐变（可折叠）：开关 + CurveField + 快捷预设（淡入/淡出/两端/全不透明）
  - 大小渐变（可折叠）：开关 + CurveField + 快捷预设（中间大/中间小/由小变大/统一）
  - minSize 调整为 400x600

### 第三轮：旋转升级为三轴曲线 + 扭曲功能
- 移除 `perInstanceRotation` Vector3，改为三条 AnimationCurve：
  - `rotationXOverPath` (Pitch 俯仰) / `rotationYOverPath` (Yaw 偏航) / `rotationZOverPath` (Roll/扭曲)
  - 每条曲线独立控制沿路径的 Euler 角度变化
  - Z轴曲线即扭曲(twist)功能：绕路径切线方向扭转
  - 曲线旋转叠加在路径朝向(alignToPath)之上
- Window新增：
  - 旋转设置默认展开（含扭曲标识），自旋速度独立保留
  - 三轴曲线各用不同颜色区分 (红/绿/蓝)
  - 扭曲快捷预设：螺旋1圈/2圈/反向1圈/零扭曲
  - 综合预设：全轴归零
  - minSize 调整为 420x650
- 文档更新：增加扭曲功能说明和常见问题
- 第五轮：补回 perInstanceRotation(Vector3)固定偏移，旋转公式：path×fixed×curve×spin

### 第六轮：修复透明效果无效
- 问题根因：只检查 `_Color` 属性（Built-in RP），URP/HDRP 使用 `_BaseColor`
- 修复：
  - `SetMaterialAlpha()` 静态方法：优先 `_Color`，回退 `_BaseColor`，使用 SetColor/GetColor
  - 生成时预创建材质实例并缓存到 `instanceMaterials`，避免每帧 `r.material` 反复实例化
  - 关闭透明度渐变时自动调用 `ResetAlpha()` 恢复 Alpha=1
  - ClearInstances 同步清理 instanceMaterials

### 第七轮：创建路径阵列动画烘焙器
- 创建 `PathArrayBaker.cs`（Editor窗口，`Moshi.PathTool.Editor`命名空间）
  - 输入：PathArrayCycler 组件引用
  - 参数：动画名前缀、持续时间、采样率(FPS)、速度曲线、输出目录
  - 烘焙内容：每个实例生成独立 .anim，包含 localPosition/localRotation/localScale
  - 旋转完全复现阵列器逻辑：pathRot × fixedRot × curveRot × spinRot
  - 缩放：如果启用了大小渐变则烘焙 scaleOverPath 曲线值
  - 模拟 PingPong 往返模式相位计算
  - 输出为本地坐标（InverseTransformPoint/Inverse rotation）
- 菜单入口：工具→Moshi→路径工具→路径阵列动画烘焙 (priority 101)
