# 图片转面片工具 (Moshi_ImgToQuad)

## 功能概述
将多张图片批量生成材质球和 Quad/Sprite/粒子系统子对象，支持从 PSD 文件导入图层位置信息。

## 打开方式
**菜单路径**: `工具 > Moshi > 图片转面片`

## 基础设置
- **目标对象**: 当前 Hierarchy 选中的对象
- **父对象名**: 生成的父级 GameObject 名称
- **Shader**: 用于生成材质的 Shader（默认使用熊猫 Shader）
- **材质文件夹**: 生成材质的保存路径

## PSD 导入功能

### 支持的格式
- PSD 文件（直接拖入工具窗口）

### PSD 导入设置
- **原点模式**: Center（中心）/ TopLeft（左上）/ BottomLeft（左下）
- **只导入可见图层**: 跳过隐藏图层
- **跳过图层组**: 不导入组节点
- **贴图输出路径**: 从 PSD 导出的贴图保存位置

### 导入流程
1. 将 PSD 文件拖入"PSD 导入"区域
2. 查看解析出的图层列表
3. 勾选要导入的图层
4. 点击"导入选中图层"

## 生成设置

### 生成模式
- **Quad**: 生成 Quad 网格对象
- **SpriteRenderer**: 生成 Sprite 渲染器对象
- **ParticleSystem**: 生成粒子系统对象

### Transform 类型
- **Transform3D**: 使用标准 3D Transform
- **RectTransform2D**: 使用 UI RectTransform

### 尺寸模式
- **像素比例**: 按像素大小缩放
- **宽度为1**: 统一宽度为 1
- **高度为1**: 统一高度为 1

### 像素单位
设置像素到世界单位的换算比例。

## 生成列表
- 显示待生成的图片列表
- 可勾选/取消选择
- 支持拖拽添加图片
- 显示 PSD 位置信息（如有）

## 使用流程
1. 在 Hierarchy 选中目标父对象
2. 配置基础设置（Shader、材质文件夹等）
3. 添加图片（拖拽或从 PSD 导入）
4. 配置生成模式和尺寸设置
5. 点击"生成对象"

## 注意事项
1. PSD 导入会自动解析图层位置
2. 材质会自动创建并保存到指定文件夹
3. 支持批量处理多张图片
4. 生成的对象会自动设置正确的位置和缩放
