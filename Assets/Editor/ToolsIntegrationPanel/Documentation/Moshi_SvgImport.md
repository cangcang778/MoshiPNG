# SVG位置导入工具 (Moshi_SvgImport)

## 功能概述
从 SVG 文件解析元素位置信息，批量生成 Quad/Sprite/粒子系统对象，适合从设计软件导出的矢量布局。

## 打开方式
**菜单路径**: `工具 > Moshi > SVG位置导入`

## 基础设置
- **目标对象**: 当前 Hierarchy 选中的对象
- **父对象名**: 生成的父级 GameObject 名称
- **Shader**: 用于生成材质的 Shader
- **材质文件夹**: 生成材质的保存路径

## SVG 导入功能

### 支持的元素类型
- `<rect>`: 矩形
- `<circle>`: 圆形
- `<ellipse>`: 椭圆
- `<image>`: 图片引用
- `<path>`: 路径（解析边界框）
- `<g>`: 分组

### 导入方式
1. **拖拽 SVG 文件**: 直接拖入工具窗口
2. **粘贴 SVG 代码**: 点击"粘贴SVG代码"按钮

### 图片目录设置
导入前需设置图片目录，用于匹配 SVG 中引用的图片资源。

### 贴图匹配模式
- **按 ID 匹配**: 根据元素 ID 在图片目录中查找同名贴图
- **按文件夹匹配**: 自动匹配文件夹中的图片
- **手动指定**: 手动为每个元素指定贴图

## 生成设置

### 生成模式
- **Quad**: 生成 Quad 网格对象
- **SpriteRenderer**: 生成 Sprite 渲染器对象
- **ParticleSystem**: 生成粒子系统对象

### Transform 类型
- **Transform3D**: 使用标准 3D Transform
- **RectTransform2D**: 使用 UI RectTransform

### 原点模式
- **Center**: 画布中心为原点
- **TopLeft**: 左上角为原点
- **BottomLeft**: 左下角为原点

### 尺寸模式
- **像素比例**: 按 SVG 中的像素大小
- **宽度为1**: 统一宽度为 1
- **高度为1**: 统一高度为 1

## 元素列表
- 显示解析出的 SVG 元素
- 可勾选/取消选择
- 显示元素类型、位置、尺寸
- 预览匹配的贴图

## 使用流程
1. 设置图片目录
2. 导入 SVG 文件
3. 配置贴图匹配模式
4. 选择要生成的元素
5. 配置生成模式和尺寸设置
6. 点击"导入选中元素"

## 注意事项
1. SVG 坐标系 Y 轴向下，工具会自动转换
2. 确保图片目录中有对应的贴图资源
3. 复杂路径只解析边界框
4. 支持多个 SVG 文件同时导入
