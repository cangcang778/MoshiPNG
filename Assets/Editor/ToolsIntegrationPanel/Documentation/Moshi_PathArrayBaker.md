# 路径阵列动画烘焙

## 功能概述
将 PathArrayCycler 中所有实例的循环运动烘焙为 AnimationClip 文件。
每个实例生成一个独立的 `.anim` 文件，可直接拖入 Animator Controller 使用。

## 前置条件
1. 场景中已有 PathArrayCycler 且已生成阵列实例
2. 路径(PathCreator)有效，至少2个路径点

## 使用步骤
1. 在「路径阵列循环」工具中配置好阵列参数并生成阵列
2. 打开 **工具 → Moshi → 路径工具 → 路径阵列动画烘焙**
3. 选择场景中的 PathArrayCycler
4. 设置动画参数：
   - **动画名前缀**：生成的 clip 文件名为 "{前缀}_{序号}.anim"
   - **持续时间**：动画总时长（秒）
   - **采样率**：每秒采样帧数（10-60 FPS）
   - **速度曲线**：可调整不同时段的速度
5. 设置输出目录
6. 点击「🏗️ 烘焙动画」

## 烘焙内容
- **位置/旋转**：每个实例的完整 Transform 动画（本地坐标）
- **缩放**：如果阵列启用了大小渐变，一并烘焙
- **旋转曲线/自旋/扭曲**：全部包含在烘焙的旋转通道中
- **透明度/颜色**：不包含（AnimationClip 不记录材质属性）

## 生成的 Clip 使用
生成的 `.anim` 文件可：
- 拖入 Animator Controller 作为状态
- 配合 Animation 组件直接播放
- 在 Timeline 中使用

## 常见问题
**Q: 没有实例可选？** 先在「路径阵列循环」工具中点击「生成阵列」。
**Q: 烘焙后旋转不对？** 检查阵列的旋转曲线和偏移旋转设置。
**Q: 生成的 clip 太多？** 可适当降低实例数量。
