# 路径工具 (Moshi_PathTool)

## 功能概述
用于从 3ds Max 导入路径数据、创建路径跟随动画，以及将路径运动烘焙为 AnimationClip。

## 打开方式
**菜单路径**: `工具 > Moshi > 路径工具 > ...`

## 功能模块

### 1. 路径导入器
**菜单**: `工具 > Moshi > 路径工具 > 路径导入器`

#### 支持的格式
- **FBX**: 将 3ds Max 样条线转为网格后导出
- **CSV**: 使用 MaxScript 导出的点坐标数据
- **JSON**: 使用 MaxScript 导出的结构化数据

#### 导入设置
- **缩放**: 导入时的缩放比例
- **坐标轴转换**: 3ds Max 到 Unity 的坐标转换
- **闭合路径**: 是否自动闭合首尾

#### 使用流程
1. 在 3ds Max 中创建样条线
2. 转为网格导出 FBX，或使用 MaxScript 导出 CSV/JSON
3. 打开路径导入器
4. 拖入导出的文件
5. 配置导入参数
6. 点击"预览"查看效果
7. 点击"导入"生成 PathCreator

### 2. 动画烘焙器
**菜单**: `工具 > Moshi > 路径工具 > 动画烘焙器`

#### 功能说明
将物体沿路径移动的轨迹烘焙为 AnimationClip 动画。

#### 烘焙设置
- **路径**: 选择 PathCreator 组件
- **目标物体**: 要烘焙动画的 GameObject
- **时长**: 动画总时长
- **采样率**: 每秒采样帧数
- **循环**: 是否设置为循环动画

#### 烘焙选项
- **位置**: 烘焙位置曲线
- **旋转**: 烘焙旋转曲线
- **沿路径朝向**: 物体朝向路径切线方向

#### 使用流程
1. 创建或导入路径
2. 打开动画烘焙器
3. 设置路径和目标物体
4. 配置烘焙参数
5. 点击"烘焙"生成 AnimationClip

### 3. 路径跟随组件
**菜单**: `工具 > Moshi > 路径工具 > 为选中物体添加路径跟随`

#### PathFollower 组件
运行时组件，用于让物体沿路径移动。

#### 参数设置
- **路径**: 引用的 PathCreator
- **速度**: 移动速度
- **跟随模式**: 一次/循环/往返
- **朝向模式**: 沿路径方向/固定朝向

#### 运行时 API
```csharp
pathFollower.Play();      // 开始跟随
pathFollower.Pause();     // 暂停
pathFollower.Stop();      // 停止
pathFollower.SetProgress(0.5f);  // 设置进度 (0-1)
```

## MaxScript 导出脚本
工具内置了 MaxScript 脚本，可在 3ds Max 中导出样条线数据。

### 获取脚本
在路径导入器窗口点击"显示导出脚本"按钮。

### 使用方法
1. 复制脚本到 3ds Max 的 MaxScript 编辑器
2. 选中要导出的样条线
3. 运行脚本
4. 选择导出格式和保存路径

## 注意事项
1. 3ds Max 使用 Z-up，Unity 使用 Y-up，导入时会自动转换
2. FBX 导入需要先在 Max 中将样条线转为可编辑网格
3. 路径点数量影响平滑度和性能
4. 烘焙的动画可直接用于 Animator 或 Timeline
