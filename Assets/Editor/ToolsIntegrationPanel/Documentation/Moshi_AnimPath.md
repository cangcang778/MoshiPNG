# 动画路径修复 (Moshi_AnimPath)

## 功能概述
批量修改 AnimationClip 和 Timeline 中的动画路径，支持替换、添加前后缀、插入、删除路径段等多种编辑模式。

## 打开方式
**菜单路径**: `工具 > Moshi > 动画路径修复`

---

## 标签页

### 1. 扫描修复
主要工作区，进行路径扫描和修改操作。

### 2. 操作日志
记录所有修改操作的历史日志。

### 3. 设置
工具参数配置。

---

## 扫描模式
- **选中对象**: 仅扫描当前选中的对象
- **单文件夹**: 扫描指定文件夹中的资源
- **多文件夹**: 同时扫描多个文件夹

## 资源类型
- **AnimClip**: 仅扫描 AnimationClip 文件
- **Timeline**: 仅扫描 Timeline 资源
- **全部**: 同时扫描两种类型

---

## 路径编辑模式

| 模式 | 说明 |
|------|------|
| 替换 | 将路径中的指定字符串替换为新字符串 |
| 添加前缀 | 在路径前添加指定前缀 |
| 添加后缀 | 在路径后添加指定后缀 |
| 插入 | 在路径指定位置插入字符串 |
| 删除段 | 删除路径中的指定层级段 |

---

## 使用流程
1. 选择扫描模式和资源类型
2. 点击"扫描"查找所有动画路径
3. 在扫描结果中查看路径列表
4. 选择路径编辑模式
5. 配置修改参数
6. 预览修改结果
7. 点击"执行修复"应用修改

---

## 注意事项
1. 修改操作支持 Undo（Ctrl+Z）
2. 建议修改前先查看预览确认
3. 同时支持 AnimationClip 和 Timeline 中的路径
4. 操作日志可追溯历史修改
