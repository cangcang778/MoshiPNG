# 粒子排序修改工具 (Moshi_ParticleOrder)

## 功能概述
用于批量增量修改粒子系统的 Order in Layer、Sorting Fudge、Start Delay 和 Start Lifetime 参数。与直接设置固定值不同，此工具在原有值基础上进行加减法运算。

## 打开方式
**菜单路径**: `工具 > Moshi > 粒子排序修改`

## 可修改参数

### 1. Order in Layer 增量修改
修改粒子系统渲染器的排序层级。

**排序模式**:
- **统一偏移**: 所有粒子系统增加/减少相同的偏移量
- **递增排序**: 按 Hierarchy 顺序依次累加步长值
- **递减排序**: 按 Hierarchy 顺序依次累减步长值

### 2. Sorting Fudge 增量修改
修改粒子系统渲染器的排序微调值。

**排序模式**: 同 Order in Layer

### 3. Start Delay 增量修改
修改粒子系统的启动延迟时间。

**排序模式**: 同上（结果不会小于 0）

### 4. Start Lifetime 增量修改
修改粒子系统的粒子生命周期。

**排序模式**: 同上（结果不会小于 0.001）

## 预览功能
- 启用"显示修改预览"可在执行前查看修改效果
- 点击"刷新预览"更新预览内容
- 预览中会用黄色高亮显示将要修改的值

## 使用示例

### 场景1：统一增加排序层级
1. 选择多个粒子系统
2. 启用 Order in Layer 修改
3. 选择"统一偏移"模式
4. 设置偏移量为 10
5. 执行后所有粒子系统的 Order in Layer 都会 +10

### 场景2：创建递增的延迟效果
1. 选择多个粒子系统
2. 启用 Start Delay 修改
3. 选择"递增排序"模式
4. 设置步长为 0.1
5. 执行后第一个 +0.1，第二个 +0.2，依此类推

## 快捷操作
- **重置所有参数**: 将所有设置恢复为默认值

## 注意事项
1. 粒子系统按 Hierarchy 中的顺序处理
2. 支持 Undo 操作（Ctrl+Z）
3. 修改后会自动标记场景为已修改状态
