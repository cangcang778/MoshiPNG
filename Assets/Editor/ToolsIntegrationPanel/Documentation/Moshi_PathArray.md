# 路径阵列循环

## 功能概述
将模型/特效沿路径均匀排列，并驱动所有对象沿路径循环运动。
适用于环形传送带、旋转展示、粒子轨道、消散效果、螺旋扭曲等场景。

## 使用步骤
1. 打开 **工具 → Moshi → 路径工具 → 路径阵列循环**
2. 在「路径设置」中指定场景中的 PathCreator
3. 在「阵列配置」中拖入预制体，设置实例数量
4. 调整「运动设置」的速度和模式、朝向
5. 展开「旋转设置（含扭曲）」配置三轴旋转曲线和自旋
6. 展开「透明渐变」用曲线控制沿路径的透明度变化
7. 展开「大小渐变」用曲线控制沿路径的缩放变化
8. 点击「🏗️ 生成阵列」
9. 使用「▶ 播放」在编辑器中预览循环效果
10. 运行时自动播放（playOnStart=true）

## 运动模式
- **Loop** — 持续循环，到达终点后无缝回到起点（推荐）
- **PingPong** — 来回往返，到终点后反向运动
- **Manual** — 手动控制相位滑块

## 旋转设置（含扭曲）
- **自旋速度** — 每秒旋转度数（绕自身Z轴），正值逆时针
- **启用旋转曲线** — 用三条独立曲线控制沿路径的三轴旋转
  - **X轴 (Pitch)** — 俯仰角沿路径变化
  - **Y轴 (Yaw)** — 偏航角沿路径变化
  - **Z轴 (Roll/扭曲)** — 绕路径切线方向扭转实例（扭曲功能）
- 扭曲快捷预设：螺旋1圈 / 螺旋2圈 / 反向螺旋1圈 / 零扭曲
- 所有曲线横轴=路径位置(0起点→1终点)，纵轴=角度(度)
- 曲线旋转叠加在路径朝向之上

## 透明渐变
- 透明度曲线：横轴=路径位置，纵轴=Alpha
- 快捷预设：淡入 / 淡出 / 两端淡入淡出 / 全不透明

## 大小渐变
- 缩放曲线：横轴=路径位置，纵轴=缩放倍率
- 快捷预设：中间大 / 中间小 / 由小变大 / 统一大小

## 常见问题
**Q: 路径不闭合？** 建议使用闭合路径(closePath=true)，循环效果更自然。
**Q: 扭曲不生效？** 需要先开启「启用旋转曲线」，Z轴曲线控制扭曲量。
**Q: 透明渐变不生效？** 素材需要有带 _Color 属性的材质。粒子系统自动使用 startColor。
**Q: 修改参数后实例没更新？** 重新点击「生成阵列」。
