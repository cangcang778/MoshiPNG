# 资源收藏夹 (Moshi_Favorites)

## 功能概述
收藏夹工具用于快速收藏和定位 Unity 项目中的资源文件和 Hierarchy 中的 GameObject，提高工作效率。

## 打开方式
**菜单路径**: `工具 > Moshi > 资源收藏夹`

## 功能说明

### 1. Project 收藏（左侧面板）
用于收藏 Project 窗口中的资源文件和文件夹。

**添加收藏的方式**:
- 在 Project 窗口选中资源后，点击"添加选中"按钮
- 直接从 Project 窗口拖拽资源到收藏面板

**操作功能**:
- **单击条目**: 选中并定位到该资源
- **右键菜单**: 
  - 选中并定位
  - 复制路径
  - 编辑（修改名称、备注、资源引用）
  - 上移/下移
  - 移除

### 2. Hierarchy 收藏（右侧面板）
用于收藏场景中的 GameObject 对象。

**添加收藏的方式**:
- 在 Hierarchy 窗口选中对象后，点击"添加选中"按钮
- 直接从 Hierarchy 窗口拖拽对象到收藏面板
- 右键菜单: `GameObject > 收藏 > 收藏选中对象到 Hierarchy 收藏`

**特点**:
- 记录对象所属场景和层级路径
- 跨场景定位：点击后会提示是否打开对应场景
- 场景保存后才能正常使用

### 3. 通用功能
- **搜索**: 顶部搜索框可快速过滤收藏项
- **清理失效**: 自动移除已删除或丢失的收藏项
- **定位配置**: 在 Project 中定位配置文件
- **分栏调整**: 拖动中间分隔线调整左右面板宽度

## 注意事项
1. Hierarchy 收藏需要先保存场景才能正常工作
2. 收藏数据保存在 `Moshi_FavoritesConfig.asset` 配置文件中
3. 支持 Undo 操作
