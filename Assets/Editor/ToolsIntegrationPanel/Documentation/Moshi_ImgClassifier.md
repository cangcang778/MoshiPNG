# 图片分类归档 (Moshi_ImgClassifier)

## 功能概述
基于视觉指纹（画面记忆）的图片分类归档工具，支持自动学习样本并批量分类图片到对应文件夹，同时集成重复图片查找与清理功能。

## 打开方式
**菜单路径**: `工具 > Moshi > 图片分类归档`

---

## 标签页

### 1. 分类归档 (Classify)
核心功能，基于视觉指纹进行图片分类。

#### 工作流程
1. **学习样本**: 指定样本文件夹，工具自动提取每张图片的 81 维视觉指纹
2. **扫描目标**: 指定待分类的图片文件夹（支持工程内和工程外路径）
3. **自动匹配**: 使用余弦相似度比较指纹，将待分类图片匹配到最相似的分类
4. **归档执行**: 将匹配的图片移动到对应的分类文件夹

#### 视觉指纹系统 (v4.0)
- 81 维特征向量：全局特征、空间分布、Alpha 空间、颜色直方图、形状特征等
- 8 维语义特征（v4.0 新增）：颜色温度、暖色比、冷色比、尖角度等
- 余弦相似度比较：比逐像素比较快 75 倍、存储小 100 倍

#### 扫描范围
- **工程内文件夹**: 使用 AssetDatabase 管理
- **工程外文件夹**: 使用文件系统直接操作，不触发 Unity 资源导入

### 2. 重复查清 (DuplicateClean)
查找并清理重复图片。

#### 功能说明
- 基于视觉指纹检测内容相似的重复图片
- 支持设置相似度阈值
- 预览重复图片对比
- 批量删除重复项

---

## 学习数据管理

### 导出数据
将已学习的分类指纹数据导出为 JSON 文件，方便备份和跨项目共享。

### 导入数据
从 JSON 文件导入学习数据，支持合并和覆盖两种模式。

> 注意：v4.0 数据与 v3.0 数据维度不兼容，旧版本数据需重新学习。

---

## 使用流程

### 分类归档
1. 添加分类（指定分类名称和样本文件夹）
2. 点击"学习样本"提取视觉指纹
3. 设置待扫描的图片文件夹
4. 点击"扫描分类"进行匹配
5. 查看匹配结果，确认无误
6. 点击"执行归档"移动文件

### 重复查清
1. 指定扫描文件夹
2. 设置相似度阈值
3. 点击扫描
4. 查看重复图片分组
5. 选择要删除的重复项
6. 执行清理

---

## 注意事项
1. 学习样本越多，分类准确率越高
2. 工程外文件归档不会触发 Unity 资源导入扫描
3. 导入/导出数据需注意版本兼容性（当前 v4.0）
4. 建议定期导出学习数据备份
5. 相似度阈值建议保持默认值，根据实际效果微调
