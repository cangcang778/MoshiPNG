# TL批量导入 (Moshi_TLBatch)

## 功能概述
Timeline Clip 批量导入工具，支持将多个素材按照指定的排列模式批量导入到 Timeline 轨道中。

## 打开方式
**菜单路径**: `工具 > Moshi > TL批量导入`

---

## 排列模式

| 模式 | 说明 |
|------|------|
| 阶梯下行 | 每个 Clip 独立轨道，时间递增，从上到下排列 |
| 阶梯上行 | 每个 Clip 独立轨道，时间递增，从下到上排列 |
| 单轨序列 | 所有 Clip 在同一轨道，首尾相接 |
| 单轨重叠 | 所有 Clip 在同一轨道，可配置时间间隔 |

---

## 使用流程
1. 在场景中选中 PlayableDirector 对象
2. 打开 TL批量导入窗口
3. 拖入要导入的素材（动画/音频/视频等）
4. 选择排列模式
5. 配置时间参数（起始时间、间隔等）
6. 点击"导入"执行

---

## 参数配置
- **起始时间**: Clip 排列的起始时间点
- **时间间隔**: 相邻 Clip 之间的时间间隔
- **Clip 时长**: 每个 Clip 的默认时长

---

## 注意事项
1. 操作支持 Undo（Ctrl+Z）
2. 需要先在场景中选中 PlayableDirector
3. Timeline 窗口需要处于打开状态
4. 阶梯模式会自动创建新轨道
5. 单轨模式在已有轨道上追加 Clip
