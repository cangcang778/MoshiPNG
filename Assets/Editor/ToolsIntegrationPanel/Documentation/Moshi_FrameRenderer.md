# 逐帧渲染器

## 概述

逐帧推进动画/粒子时间轴，每帧截图输出 PNG 序列帧。适用于模型转场动画、粒子特效等需要导出帧序列的场景。

> 类似 3ds Max 的逐帧渲染流程。

---

## 使用步骤

1. **准备场景**: 先用「渲染场景生成器」创建渲染环境，拖入模型/特效
2. 打开 `工具/Moshi/逐帧渲染`
3. 确认自动检测到渲染相机，或手动指定
4. 指定渲染目标物体（可选，用于采样其动画/粒子）
5. 设置帧范围、FPS、分辨率、超采样等参数
6. 配置输出目录和文件名前缀
7. 可使用预览功能检查效果
8. 点击「🎬 开始逐帧渲染」

---

## 渲染参数

| 参数 | 说明 |
|---|---|
| **帧范围** | 起始帧 ~ 结束帧，自动从 Animator/Animation 检测 |
| **FPS** | 输出帧率，控制帧间时间步进 |
| **分辨率** | 可使用相机设置或自定义 |
| **超采样** | 1x/2x/4x，渲染更大再缩放，提升边缘质量 |
| **背景模式** | 使用相机设置 / 强制覆盖 / 强制透明 |
| **输出目录** | 默认 `Assets/Moshi_FrameOutput/` |

---

## 预览功能

- 拖动时间轴滑块预览任意帧
- 逐帧步进按钮 (◀◀ ◀ ▶ ▶▶)
- 循环播放模式，在 Scene View 中实时预览
- 当前帧缩略图显示

---

## 图集打包

点击「📦 打包为图集」可在渲染完成后自动将所有帧拼成一张 Sprite Sheet，方便用于 2D 游戏或 Spine/DragonBones。

---

## 支持的目标类型

- **Animator**: 自动采样 AnimationClip 到指定时间
- **Animation**: 传统 Animation 组件
- **ParticleSystem**: 使用 Simulate() 逐帧推进粒子
- **无目标**: 渲染当前场景中所有粒子系统

---

## 输出文件

```
Assets/Moshi_FrameOutput/
├── Frame_0000.png
├── Frame_0001.png
├── Frame_0002.png
├── ...
└── Frame_SpriteSheet.png  (图集模式)
```
